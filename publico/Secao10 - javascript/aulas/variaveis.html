JAVASCRIPT TEM TIPO!

No JavaScript a tipagem é dinâmica, por isso não é necessário declarar o tipo de uma variável, basta atribuir seu valor e a linguagem assume se é uma sequência de caracteres, números, indefinido ou outras

Principais tipos:

  • string
  • number - interio o fracionário
  • boolean

Regras para declaração

  • Não podem ser iniciadas com numeros, apenas letras ou "_"
  • Não podemos utilizar caracters especiais
  • Não podem ser palavras reservadas

É recomendado utilizar a expressão "var" antes de declarar, pois fica mais fácil tanto para o interpretado quando para nós

Case Sensitive

null é diferente de undefined
