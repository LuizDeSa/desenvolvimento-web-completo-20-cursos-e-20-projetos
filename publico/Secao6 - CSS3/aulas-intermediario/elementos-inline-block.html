Elementos block -> Ocupa todo espaço (largura) da tela disponivel, fica um ebaixo do outro. Ex: h1, p, table....

Elementos inline -> largura da caixa do elemento é de acordo com o seu conteúdo. Fica um elemento a frente do outro.Ex: a, span, img ...

Elementos inline-block -> largura definida pelo seu conteúdo mas fica um elemento abaixo do outro quando não tem mais espaço na frente. Ex: utilizando a propriedade display



É possivel mudar essas caracteristicas por meio do css
