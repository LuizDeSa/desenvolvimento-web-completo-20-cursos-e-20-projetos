Posicionamentos

static -> O fluxo padrão é elemento do tipo block um abaixo do outro

relative -> Ao utilizar é necessário utilizar outras propriedades de deslocamentos: top, right, bottom, left

absolute -> O elemento sai do fluxo normal do documento e os outros elementos se comportam coomo se esse elemento não existisse. Ele será sempre posiscionado baseado no container que ele está dentro, desde que este container tenha posicionamento diferente do padrão(static)

Fixed -> O elemento fica fixo em relação a tela do navegador
